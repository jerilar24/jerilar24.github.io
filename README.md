# Jeremmy Leiva Portfolio

A simple static portfolio site for academic background, work experience,
projects, and contact information.

## Edit Content

- Update project descriptions and links in `index.html` as more details become available.
- Hero image: `assets/hero-workspace.png`.

## Local Preview

Open `index.html` directly in a browser, or run a static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deployment

GitHub Pages, plain HTML, CSS, and
JavaScript with no build step.
