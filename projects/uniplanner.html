Mobile app

UniPlanner Student Planner App

An Android app for students to plan homework, check class schedules, contact professors, and track attendance.

My role

Backend and database development

I developed the backend logic and database layer for UniPlanner using Java and SQLite. The app was designed around everyday student needs: keeping assignments organized, reviewing schedules, reaching professors, and tracking attendance from one place.

The main technical focus was building a reliable local data model for academic information and connecting it cleanly with the Android app experience.
